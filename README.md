# Cloud Infrastructure Automation

This project demonstrates a production-ready Infrastructure as Code (IaC) implementation to provision and manage cloud infrastructure using Terraform.

The goal of this project is to automate infrastructure deployment, ensure consistency across environments, and follow DevOps best practices for scalable and secure cloud architecture.
